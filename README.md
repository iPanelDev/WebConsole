# iPanel WebConsole

iPanel系列的前端网页代码，基于Vue3

- 文档：<https://ipaneldev.github.io/docs/guide/webConsole/intro>
- Powered by [admin-one-vue-tailwind](https://github.com/justboil/admin-one-vue-tailwind)

## 安装依赖

```sh
npm install
```

### 开发

```sh
npm run dev
```

### 构建

```sh
npm run build
```
